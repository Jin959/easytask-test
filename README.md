## 개요

대규모 매칭은 어떻게 이루어질까?

인증과 인가는 어떤 방식으로 동작하는 것일까?

기존 서비스를 고도화 하는 건 어떻게 진행되는 것일까?

이러한 궁금증들을 통해서 직접 이지테스크 서버를 구현해보는 프로젝트를 진행하게 되었습니다.

<br>

---

## 사용기술 및 환경

- Java 11
- Spring Boot
- MySQL 8.0
- MyBatis
- JPA
- Redis
- JWT
